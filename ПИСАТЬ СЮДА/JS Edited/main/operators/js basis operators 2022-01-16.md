## [–°–ª–æ–∂–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫ –ø—Ä–∏ –ø–æ–º–æ—â–∏ –±–∏–Ω–∞—Ä–Ω–æ–≥–æ +](https://learn.javascript.ru/operators#slozhenie-strok-pri-pomoschi-binarnogo)

## [–ü—Ä–∏–≤–µ–¥–µ–Ω–∏–µ –∫ —á–∏—Å–ª—É, —É–Ω–∞—Ä–Ω—ã–π +](https://learn.javascript.ru/operators#privedenie-k-chislu-unarnyy)

–ü–ª—é—Å `+` —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –≤ –¥–≤—É—Ö —Ñ–æ—Ä–º–∞—Ö: –±–∏–Ω–∞—Ä–Ω–æ–π, –∫–æ—Ç–æ—Ä—É—é –º—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª–∏ –≤—ã—à–µ, –∏ —É–Ω–∞—Ä–Ω–æ–π.

## [–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç](https://learn.javascript.ru/operators#inkrement-dekrement)

-   **–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç** `++` —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –Ω–∞ 1:
    

**–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç –º–æ–∂–Ω–æ –ø—Ä–∏–º–µ–Ω–∏—Ç—å —Ç–æ–ª—å–∫–æ –∫ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π. –ü–æ–ø—ã—Ç–∫–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –µ–≥–æ –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏–∏, —Ç–∏–ø–∞ 5++, –ø—Ä–∏–≤–µ–¥—ë—Ç –∫ –æ—à–∏–±–∫–µ.**

–û–ø–µ—Ä–∞—Ç–æ—Ä—ã `++` –∏ `--` –º–æ–≥—É—Ç –±—ã—Ç—å —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω—ã –Ω–µ —Ç–æ–ª—å–∫–æ –ø–æ—Å–ª–µ, –Ω–æ –∏ –¥–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π.

-   –ö–æ–≥–¥–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–¥—ë—Ç –ø–æ—Å–ª–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π ‚Äî —ç—Ç–æ ¬´–ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞¬ª: `counter++`.
-   ¬´–ü—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞¬ª ‚Äî —ç—Ç–æ –∫–æ–≥–¥–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–¥—ë—Ç –ø–µ—Ä–µ–¥ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π: `++counter`.

–û–±–µ —ç—Ç–∏ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –¥–µ–ª–∞—é—Ç –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ: —É–≤–µ–ª–∏—á–∏–≤–∞—é—Ç `counter` –Ω–∞ `1`.

–ï—Å—Ç—å –ª–∏ —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É –Ω–∏–º–∏? –î–∞, –Ω–æ —É–≤–∏–¥–µ—Ç—å –µ—ë –º—ã —Å–º–æ–∂–µ–º, —Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –±—É–¥–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç `++/--`.

–î–∞–≤–∞–π—Ç–µ –ø—Ä–æ—è—Å–Ω–∏–º —ç—Ç–æ—Ç –º–æ–º–µ–Ω—Ç. –ö–∞–∫ –º—ã –∑–Ω–∞–µ–º, –≤—Å–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∑–Ω–∞—á–µ–Ω–∏–µ. –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –∏–Ω–∫—Ä–µ–º–µ–Ω—Ç–∞/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç–∞ –Ω–µ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ. –ü—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –≤ —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ç–∞—Ä–æ–µ (–¥–æ —É–≤–µ–ª–∏—á–µ–Ω–∏—è/—É–º–µ–Ω—å—à–µ–Ω–∏—è —á–∏—Å–ª–∞).


–í —Å—Ç—Ä–æ–∫–µ `(*)` _–ø—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è_ —Ñ–æ—Ä–º–∞ `++counter` —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç `counter` –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `2`. –¢–∞–∫ —á—Ç–æ `alert` –ø–æ–∫–∞–∂–µ—Ç `2`.

–¢–µ–ø–µ—Ä—å –ø–æ—Å–º–æ—Ç—Ä–∏–º –Ω–∞ –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω—É—é —Ñ–æ—Ä–º—É:
```javascript

let a = counter++; // (*) –º–µ–Ω—è–µ–º ++counter –Ω–∞ counter++

```

–í —Å—Ç—Ä–æ–∫–µ `(*)` _–ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è_ —Ñ–æ—Ä–º–∞ `counter++` —Ç–∞–∫–∂–µ —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç `counter`, –Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç _—Å—Ç–∞—Ä–æ–µ_ –∑–Ω–∞—á–µ–Ω–∏–µ (–∫–æ—Ç–æ—Ä–æ–µ –±—ã–ª–æ –¥–æ —É–≤–µ–ª–∏—á–µ–Ω–∏—è). –¢–∞–∫ —á—Ç–æ `alert` –ø–æ–∫–∞–∂–µ—Ç `1`.

–ü–æ–¥–≤–µ–¥—ë–º –∏—Ç–æ–≥–∏:

-   –ï—Å–ª–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è, –∞ –Ω—É–∂–Ω–æ —Ç–æ–ª—å–∫–æ —É–≤–µ–ª–∏—á–∏—Ç—å/—É–º–µ–Ω—å—à–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é, —Ç–æ–≥–¥–∞ –±–µ–∑ —Ä–∞–∑–Ω–∏—Ü—ã, –∫–∞–∫—É—é —Ñ–æ—Ä–º—É –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:

-   –ï—Å–ª–∏ —Ö–æ—á–µ—Ç—Å—è —Ç—É—Ç –∂–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç, —Ç–æ –Ω—É–∂–Ω–∞ –ø—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞:

    
-   –ï—Å–ª–∏ –Ω—É–∂–Ω–æ —É–≤–µ–ª–∏—á–∏—Ç—å –∏ –ø—Ä–∏ —ç—Ç–æ–º –ø–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π _–¥–æ —É–≤–µ–ª–∏—á–µ–Ω–∏—è_ ‚Äì –Ω—É–∂–Ω–∞ –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞:

    
–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤ –ª—é–±—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏—è—Ö

–û–ø–µ—Ä–∞—Ç–æ—Ä—ã `++/--` –º–æ–≥—É—Ç —Ç–∞–∫–∂–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –≤–Ω—É—Ç—Ä–∏ –≤—ã—Ä–∞–∂–µ–Ω–∏–π. –ò—Ö –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –≤—ã—à–µ, —á–µ–º —É –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–∞ –¥—Ä—É–≥–∏—Ö –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π.

## [–ü–æ–±–∏—Ç–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã](https://learn.javascript.ru/operators#pobitovye-operatory)

–û–Ω–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ä–µ–¥–∫–æ, –∫–æ–≥–¥–∞ –≤–æ–∑–Ω–∏–∫–∞–µ—Ç –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å –æ–ø–µ—Ä–∏—Ä–æ–≤–∞—Ç—å —Å —á–∏—Å–ª–∞–º–∏ –Ω–∞ –æ—á–µ–Ω—å –Ω–∏–∑–∫–æ–º (–ø–æ–±–∏—Ç–æ–≤–æ–º) —É—Ä–æ–≤–Ω–µ. –í –±–ª–∏–∂–∞–π—à–µ–º –≤—Ä–µ–º–µ–Ω–∏ –æ–Ω–∏ –Ω–∞–º –Ω–µ –ø–æ–Ω–∞–¥–æ–±—è—Ç—Å—è, —Ç–∞–∫ –∫–∞–∫ –≤–µ–±-—Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∏ —Ä–µ–¥–∫–æ –∫ –Ω–∏–º –ø—Ä–∏–±–µ–≥–∞—é—Ç, —Ö–æ—Ç—è –≤ –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å—Ñ–µ—Ä–∞—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤ –∫—Ä–∏–ø—Ç–æ–≥—Ä–∞—Ñ–∏–∏) –æ–Ω–∏ –ø–æ–ª–µ–∑–Ω—ã. –ú–æ–∂–µ—Ç–µ –ø—Ä–æ—á–∏—Ç–∞—Ç—å [—Ä–∞–∑–¥–µ–ª –æ –Ω–∏—Ö](https://developer.mozilla.org/ru/docs/Web/JavaScript/Guide/Expressions_and_Operators#%D0%91%D0%B8%D1%82%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BF%D0%BE%D1%80%D0%B0%D0%B7%D1%80%D1%8F%D0%B4%D0%BD%D1%8B%D0%B5_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%8B) –Ω–∞ MDN, –∫–æ–≥–¥–∞ –≤–æ–∑–Ω–∏–∫–Ω–µ—Ç —Ä–µ–∞–ª—å–Ω–∞—è –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å.

## [–û–ø–µ—Ä–∞—Ç–æ—Ä ¬´–∑–∞–ø—è—Ç–∞—è¬ª](https://learn.javascript.ru/operators#operator-zapyataya)

–û–ø–µ—Ä–∞—Ç–æ—Ä ¬´–∑–∞–ø—è—Ç–∞—è¬ª (`,`) —Ä–µ–¥–∫–æ –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –∏ —è–≤–ª—è–µ—Ç—Å—è –æ–¥–Ω–∏–º –∏–∑ —Å–∞–º—ã—Ö –Ω–µ–æ–±—ã—á–Ω—ã—Ö. –ò–Ω–æ–≥–¥–∞ –æ–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –Ω–∞–ø–∏—Å–∞–Ω–∏—è –±–æ–ª–µ–µ –∫–æ—Ä–æ—Ç–∫–æ–≥–æ –∫–æ–¥–∞, –ø–æ—ç—Ç–æ–º—É –Ω–∞–º –Ω—É–∂–Ω–æ –∑–Ω–∞—Ç—å –µ–≥–æ, —á—Ç–æ–±—ã –ø–æ–Ω–∏–º–∞—Ç—å, —á—Ç–æ –ø—Ä–∏ —ç—Ç–æ–º –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç.

–û–ø–µ—Ä–∞—Ç–æ—Ä ¬´–∑–∞–ø—è—Ç–∞—è¬ª –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –Ω–∞–º –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –≤—ã—á–∏—Å–ª—è—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –≤—ã—Ä–∞–∂–µ–Ω–∏–π, —Ä–∞–∑–¥–µ–ª—è—è –∏—Ö –∑–∞–ø—è—Ç–æ–π `,`. –ö–∞–∂–¥–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ç–æ–ª—å–∫–æ –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ.

–ó–∞–ø—è—Ç–∞—è –∏–º–µ–µ—Ç –æ—á–µ–Ω—å –Ω–∏–∑–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç

**–ü–æ–∂–∞–ª—É–π—Å—Ç–∞, –æ–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä `,` –∏–º–µ–µ—Ç –æ—á–µ–Ω—å –Ω–∏–∑–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç, –Ω–∏–∂–µ `=`, –ø–æ—ç—Ç–æ–º—É —Å–∫–æ–±–∫–∏ –≤–∞–∂–Ω—ã –≤ –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–º –≤—ã—à–µ –ø—Ä–∏–º–µ—Ä–µ.**

–ë–µ–∑ –Ω–∏—Ö –≤ `a = 1 + 2, 3 + 4` —Å–Ω–∞—á–∞–ª–∞ –≤—ã–ø–æ–ª–Ω–∏—Ç—Å—è `+`, —Å—É–º–º–∏—Ä—É—è —á–∏—Å–ª–∞ –≤ `a = 3, 7`, –∑–∞—Ç–µ–º –æ–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è `=` –ø—Ä–∏—Å–≤–æ–∏—Ç `a = 3`, –∞ —Ç–æ, —á—Ç–æ –∏–¥—ë—Ç –¥–∞–ª—å—à–µ, –±—É–¥–µ—Ç –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞–Ω–æ. –í—Å—ë —Ç–∞–∫ –∂–µ, –∫–∞–∫ –≤ `(a = 1 + 2), 3 + 4`.

–ó–∞—á–µ–º –Ω–∞–º –æ–ø–µ—Ä–∞—Ç–æ—Ä, –∫–æ—Ç–æ—Ä—ã–π –æ—Ç–±—Ä–∞—Å—ã–≤–∞–µ—Ç –≤—Å—ë, –∫—Ä–æ–º–µ –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è?

–ò–Ω–æ–≥–¥–∞ –µ–≥–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –≤ —Å–æ—Å—Ç–∞–≤–µ –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã—Ö –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–π, —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –¥–µ–π—Å—Ç–≤–∏–π –≤ –æ–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–µ.

# –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è

-   –ë–æ–ª—å—à–µ/–º–µ–Ω—å—à–µ: `a > b`, `a < b`.
-   –ë–æ–ª—å—à–µ/–º–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ: `a >= b`, `a <= b`.
-   –†–∞–≤–Ω–æ: `a == b`. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, –¥–ª—è —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–≤–æ–π–Ω–æ–π –∑–Ω–∞–∫ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `==`. –û–¥–∏–Ω –∑–Ω–∞–∫ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `a = b` –æ–∑–Ω–∞—á–∞–ª –±—ã –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ.
-   –ù–µ —Ä–∞–≤–Ω–æ. –í –º–∞—Ç–µ–º–∞—Ç–∏–∫–µ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è —Å–∏–º–≤–æ–ª–æ–º `‚â†`, –Ω–æ –≤ JavaScript –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –∫–∞–∫ `a != b`.

**–†–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –∏–º–µ–µ—Ç –ª–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø**

–í—Å–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ —Ç–∏–ø–∞:

-   `true` ‚Äì –æ–∑–Ω–∞—á–∞–µ—Ç ¬´–¥–∞¬ª, ¬´–≤–µ—Ä–Ω–æ¬ª, ¬´–∏—Å—Ç–∏–Ω–∞¬ª.
-   `false` ‚Äì –æ–∑–Ω–∞—á–∞–µ—Ç ¬´–Ω–µ—Ç¬ª, ¬´–Ω–µ–≤–µ—Ä–Ω–æ¬ª, ¬´–ª–æ–∂—å¬ª.

–†–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –º–æ–∂–Ω–æ –ø—Ä–∏—Å–≤–æ–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –∫–∞–∫ –∏ –ª—é–±–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ:

`let result = 5 > 4; // —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π result

## [–°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫](https://learn.javascript.ru/comparison#sravnenie-strok)

–ß—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å, —á—Ç–æ –æ–¥–Ω–∞ —Å—Ç—Ä–æ–∫–∞ –±–æ–ª—å—à–µ –¥—Ä—É–≥–æ–π, JavaScript –∏—Å–ø–æ–ª—å–∑—É–µ—Ç ¬´–∞–ª—Ñ–∞–≤–∏—Ç–Ω—ã–π¬ª –∏–ª–∏ ¬´–ª–µ–∫—Å–∏–∫–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏–π¬ª –ø–æ—Ä—è–¥–æ–∫.

–î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, —Å—Ç—Ä–æ–∫–∏ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ø–æ—Å–∏–º–≤–æ–ª—å–Ω–æ.


–ê–ª–≥–æ—Ä–∏—Ç–º —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –¥–≤—É—Ö —Å—Ç—Ä–æ–∫ –¥–æ–≤–æ–ª—å–Ω–æ –ø—Ä–æ—Å—Ç:

1.  –°–Ω–∞—á–∞–ª–∞ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ø–µ—Ä–≤—ã–µ —Å–∏–º–≤–æ–ª—ã —Å—Ç—Ä–æ–∫.
2.  –ï—Å–ª–∏ –ø–µ—Ä–≤—ã–π —Å–∏–º–≤–æ–ª –ø–µ—Ä–≤–æ–π —Å—Ç—Ä–æ–∫–∏ –±–æ–ª—å—à–µ (–º–µ–Ω—å—à–µ), —á–µ–º –ø–µ—Ä–≤—ã–π —Å–∏–º–≤–æ–ª –≤—Ç–æ—Ä–æ–π, —Ç–æ –ø–µ—Ä–≤–∞—è —Å—Ç—Ä–æ–∫–∞ –±–æ–ª—å—à–µ (–º–µ–Ω—å—à–µ) –≤—Ç–æ—Ä–æ–π. –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –∑–∞–≤–µ—Ä—à–µ–Ω–æ.
3.  –ï—Å–ª–∏ –ø–µ—Ä–≤—ã–µ —Å–∏–º–≤–æ–ª—ã —Ä–∞–≤–Ω—ã, —Ç–æ —Ç–∞–∫–∏–º –∂–µ –æ–±—Ä–∞–∑–æ–º —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è —É–∂–µ –≤—Ç–æ—Ä—ã–µ —Å–∏–º–≤–æ–ª—ã —Å—Ç—Ä–æ–∫.
4.  –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç—Å—è, –ø–æ–∫–∞ –Ω–µ –∑–∞–∫–æ–Ω—á–∏—Ç—Å—è –æ–¥–Ω–∞ –∏–∑ —Å—Ç—Ä–æ–∫.
5.  –ï—Å–ª–∏ –æ–±–µ —Å—Ç—Ä–æ–∫–∏ –∑–∞–∫–∞–Ω—á–∏–≤–∞—é—Ç—Å—è –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ, —Ç–æ –æ–Ω–∏ —Ä–∞–≤–Ω—ã. –ò–Ω–∞—á–µ, –±–æ–ª—å—à–µ–π —Å—á–∏—Ç–∞–µ—Ç—Å—è –±–æ–ª–µ–µ –¥–ª–∏–Ω–Ω–∞—è —Å—Ç—Ä–æ–∫–∞.

–í –ø—Ä–∏–º–µ—Ä–∞—Ö –≤—ã—à–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ `'–Ø' > '–ê'` –∑–∞–≤–µ—Ä—à–∏—Ç—Å—è –Ω–∞ –ø–µ—Ä–≤–æ–º —à–∞–≥–µ, —Ç–æ–≥–¥–∞ –∫–∞–∫ —Å—Ç—Ä–æ–∫–∏ `'–ö–æ—Ç—ã'` –∏ `'–ö–æ–¥–∞'` –±—É–¥—É—Ç —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å—Å—è –ø–æ—Å–∏–º–≤–æ–ª—å–Ω–æ:

1.  `–ö` —Ä–∞–≤–Ω–∞ `–ö`.
2.  `–æ` —Ä–∞–≤–Ω–∞ `–æ`.
3.  `—Ç` –±–æ–ª—å—à–µ, —á–µ–º `–¥`. –ù–∞ —ç—Ç–æ–º —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è. –ü–µ—Ä–≤–∞—è —Å—Ç—Ä–æ–∫–∞ –±–æ–ª—å—à–µ.

–ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–æ–¥–∏—Ä–æ–≤–∫–∞ Unicode, –∞ –Ω–µ –Ω–∞—Å—Ç–æ—è—â–∏–π –∞–ª—Ñ–∞–≤–∏—Ç

–ü—Ä–∏–≤–µ–¥—ë–Ω–Ω—ã–π –≤—ã—à–µ –∞–ª–≥–æ—Ä–∏—Ç–º —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –ø–æ—Ö–æ–∂ –Ω–∞ –∞–ª–≥–æ—Ä–∏—Ç–º, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–π –≤ —Å–ª–æ–≤–∞—Ä—è—Ö –∏ —Ç–µ–ª–µ—Ñ–æ–Ω–Ω—ã—Ö –∫–Ω–∏–≥–∞—Ö, –Ω–æ –º–µ–∂–¥—É –Ω–∏–º–∏ –µ—Å—Ç—å –∏ —Ä–∞–∑–ª–∏—á–∏—è.

–ù–∞–ø—Ä–∏–º–µ—Ä, –≤ JavaScript –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ —Ä–µ–≥–∏—Å—Ç—Ä —Å–∏–º–≤–æ–ª–æ–≤. –ó–∞–≥–ª–∞–≤–Ω–∞—è –±—É–∫–≤–∞ `"A"` –Ω–µ —Ä–∞–≤–Ω–∞ —Å—Ç—Ä–æ—á–Ω–æ–π `"a"`. –ö–∞–∫–∞—è –∂–µ –∏–∑ –Ω–∏—Ö –±–æ–ª—å—à–µ? –°—Ç—Ä–æ—á–Ω–∞—è `"a"`. –ü–æ—á–µ–º—É? –ü–æ—Ç–æ–º—É —á—Ç–æ —Å—Ç—Ä–æ—á–Ω—ã–µ –±—É–∫–≤—ã –∏–º–µ—é—Ç –±–æ–ª—å—à–∏–π –∫–æ–¥ –≤–æ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–π —Ç–∞–±–ª–∏—Ü–µ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è, –∫–æ—Ç–æ—Ä—É—é –∏—Å–ø–æ–ª—å–∑—É–µ—Ç JavaScript (Unicode). –ú—ã –µ—â—ë –ø–æ–≥–æ–≤–æ—Ä–∏–º –æ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–º –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–∏ —Å—Ç—Ä–æ–∫ –∏ –µ–≥–æ –≤–ª–∏—è–Ω–∏–∏ –≤ –≥–ª–∞–≤–µ [–°—Ç—Ä–æ–∫–∏](https://learn.javascript.ru/string).

## [–°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤](https://learn.javascript.ru/comparison#sravnenie-raznyh-tipov)

*–ü—Ä–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–∏ –∑–Ω–∞—á–µ–Ω–∏–π —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤ JavaScript –ø—Ä–∏–≤–æ–¥–∏—Ç –∫–∞–∂–¥–æ–µ –∏–∑ –Ω–∏—Ö –∫ —á–∏—Å–ª—É.*


–õ–æ–≥–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `true` —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è `1`, –∞ `false` ‚Äì `0`.


–í–æ–∑–º–æ–∂–Ω–∞ —Å–ª–µ–¥—É—é—â–∞—è —Å–∏—Ç—É–∞—Ü–∏—è:

-   –î–≤–∞ –∑–Ω–∞—á–µ–Ω–∏—è —Ä–∞–≤–Ω—ã.
-   –û–¥–Ω–æ –∏–∑ –Ω–∏—Ö `true` –∫–∞–∫ –ª–æ–≥–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –¥—Ä—É–≥–æ–µ ‚Äì `false`.

```javascript

alert( Boolean(a) ); // false

let b = "0";

```

–° —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è JavaScript, —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–∂–∏–¥–∞–µ–º. –†–∞–≤–µ–Ω—Å—Ç–≤–æ –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –∑–Ω–∞—á–µ–Ω–∏—è, –∏—Å–ø–æ–ª—å–∑—É—è —á–∏—Å–ª–æ–≤–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ, –ø–æ—ç—Ç–æ–º—É `"0"` —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è `0`. –í —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ —è–≤–Ω–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é `Boolean` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –¥—Ä—É–≥–æ–π –Ω–∞–±–æ—Ä –ø—Ä–∞–≤–∏–ª.

## [–°—Ç—Ä–æ–≥–æ–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ](https://learn.javascript.ru/comparison#strogoe-sravnenie)

–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –æ–±—ã—á–Ω–æ–≥–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è `==` –º–æ–∂–µ—Ç –≤—ã–∑—ã–≤–∞—Ç—å –ø—Ä–æ–±–ª–µ–º—ã. –ù–∞–ø—Ä–∏–º–µ—Ä, –æ–Ω–æ –Ω–µ –æ—Ç–ª–∏—á–∞–µ—Ç `0` –æ—Ç `false`:


–¢–∞ –∂–µ –ø—Ä–æ–±–ª–µ–º–∞ —Å –ø—É—Å—Ç–æ–π —Å—Ç—Ä–æ–∫–æ–π:

–≠—Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –∏–∑-–∑–∞ —Ç–æ–≥–æ, —á—Ç–æ –æ–ø–µ—Ä–∞–Ω–¥—ã —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤ –ø—Ä–µ–æ–±—Ä–∞–∑—É—é—Ç—Å—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º `==` –∫ —á–∏—Å–ª—É. –í –∏—Ç–æ–≥–µ, –∏ –ø—É—Å—Ç–∞—è —Å—Ç—Ä–æ–∫–∞, –∏ `false` —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è –Ω—É–ª—ë–º.

–ö–∞–∫ –∂–µ —Ç–æ–≥–¥–∞ –æ—Ç–ª–∏—á–∞—Ç—å `0` –æ—Ç `false`?

**–û–ø–µ—Ä–∞—Ç–æ—Ä —Å—Ç—Ä–æ–≥–æ–≥–æ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `===` –ø—Ä–æ–≤–µ—Ä—è–µ—Ç —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –±–µ–∑ –ø—Ä–∏–≤–µ–¥–µ–Ω–∏—è —Ç–∏–ø–æ–≤.**

–î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, –µ—Å–ª–∏ `a` –∏ `b` –∏–º–µ—é—Ç —Ä–∞–∑–Ω—ã–µ —Ç–∏–ø—ã, —Ç–æ –ø—Ä–æ–≤–µ—Ä–∫–∞ `a === b` –Ω–µ–º–µ–¥–ª–µ–Ω–Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `false` –±–µ–∑ –ø–æ–ø—ã—Ç–∫–∏ –∏—Ö –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è.

–ï—â—ë –µ—Å—Ç—å –æ–ø–µ—Ä–∞—Ç–æ—Ä —Å—Ç—Ä–æ–≥–æ–≥–æ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `!==`, –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã–π `!=`.

–û–ø–µ—Ä–∞—Ç–æ—Ä —Å—Ç—Ä–æ–≥–æ–≥–æ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –¥–æ–ª—å—à–µ –ø–∏—Å–∞—Ç—å, –Ω–æ –æ–Ω –¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ –æ—á–µ–≤–∏–¥–Ω—ã–º –∏ –æ—Å—Ç–∞–≤–ª—è–µ—Ç –º–µ–Ω—å—à–µ –º–µ—Å—Ç–∞ –¥–ª—è –æ—à–∏–±–æ–∫.

## [–°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å null –∏ undefined](https://learn.javascript.ru/comparison#sravnenie-s-null-i-undefined)

–ü–æ–≤–µ–¥–µ–Ω–∏–µ `null` –∏ `undefined` –ø—Ä–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–∏ —Å –¥—Ä—É–≥–∏–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏ ‚Äî –æ—Å–æ–±–æ–µ:

–ü—Ä–∏ —Å—Ç—Ä–æ–≥–æ–º —Ä–∞–≤–µ–Ω—Å—Ç–≤–µ `===`

–ü—Ä–∏ –Ω–µ—Å—Ç—Ä–æ–≥–æ–º —Ä–∞–≤–µ–Ω—Å—Ç–≤–µ `==`
