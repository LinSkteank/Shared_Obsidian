**–í –ø—Ä–æ—Ü–µ—Å—Å–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –¥–≤–∏–∂–æ–∫ JavaScript –ø—ã—Ç–∞–µ—Ç—Å—è –Ω–∞–π—Ç–∏ –∏ –≤—ã–∑–≤–∞—Ç—å —Ç—Ä–∏ —Å–ª–µ–¥—É—é—â–∏—Ö –º–µ—Ç–æ–¥–∞ –æ–±—ä–µ–∫—Ç–∞:**

3.  –í —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ —Ö–∏–Ω—Ç —Ä–∞–≤–µ–Ω `"number"` –∏–ª–∏ `"default"`

## [Symbol.toPrimitive](https://learn.javascript.ru/object-toprimitive#symbol-toprimitive)

–ù–∞—á–Ω—ë–º —Å —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ ‚Äì —Å–∏–º–≤–æ–ª–∞ `Symbol.toPrimitive`: –º–µ—Ç–æ–¥ —Å —Ç–∞–∫–∏–º –Ω–∞–∑–≤–∞–Ω–∏–µ–º (–µ—Å–ª–∏ –µ—Å—Ç—å) –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –≤—Å–µ—Ö –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π:

`obj[Symbol.toPrimitive] = function(hint) {
  // –¥–æ–ª–∂–µ–Ω –≤–µ—Ä–Ω—É—Ç—å –ø—Ä–∏–º–∏—Ç–∏–≤–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ
  // hint —Ä–∞–≤–Ω–æ —á–µ–º—É-—Ç–æ –æ–¥–Ω–æ–º—É –∏–∑: "string", "number" –∏–ª–∏ "default"
};`

–î–ª—è –ø—Ä–∏–º–µ—Ä–∞ –∏—Å–ø–æ–ª—å–∑—É–µ–º –µ–≥–æ –≤ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –æ–±—ä–µ–∫—Ç–∞ `user`:
``````

  name: "John",
  money: 1000,

  [Symbol.toPrimitive](hint) {
    alert(`hint: ${hint}`);
    return hint == "string" ? `{name: "${this.name}"}` : this.money;
  }

// –¥–µ–º–æ–Ω—Å—Ç—Ä–∞—Ü–∏—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–≤ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π:

–ö–∞–∫ –º—ã –≤–∏–¥–∏–º –∏–∑ –∫–æ–¥–∞, `user` –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤—ã–≤–∞–µ—Ç—Å—è –ª–∏–±–æ –≤ –∏–Ω—Ñ–æ—Ä–º–∞—Ç–∏–≤–Ω—É—é —á–∏—Ç–∞–µ–º—É—é —Å—Ç—Ä–æ–∫—É, –ª–∏–±–æ –≤ –¥–µ–Ω–µ–∂–Ω—ã–π —Å—á—ë—Ç –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –∑–Ω–∞—á–µ–Ω–∏—è —Ö–∏–Ω—Ç–∞. –ï–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–π –º–µ—Ç–æ–¥ `user[Symbol.toPrimitive]` —Å–º–æ–≥ –æ–±—Ä–∞–±–æ—Ç–∞—Ç—å –≤—Å–µ —Å–ª—É—á–∞–∏ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π.

## [–ú–µ—Ç–æ–¥—ã toString/valueOf](https://learn.javascript.ru/object-toprimitive#metody-tostring-valueof)

–ú–µ—Ç–æ–¥—ã `toString` –∏ `valueOf` –±–µ—Ä—É—Ç —Å–≤–æ—ë –Ω–∞—á–∞–ª–æ —Å –¥—Ä–µ–≤–Ω–∏—Ö –≤—Ä–µ–º—ë–Ω. –≠—Ç–æ –Ω–µ —Å–∏–º–≤–æ–ª—ã, —Ç–∞–∫ –∫–∞–∫ –≤ —Ç–æ –≤—Ä–µ–º—è —Å–∏–º–≤–æ–ª–æ–≤ –µ—â—ë –Ω–µ —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–ª–æ, –∞ –ø—Ä–æ—Å—Ç–æ –æ–±—ã—á–Ω—ã–µ –º–µ—Ç–æ–¥—ã –æ–±—ä–µ–∫—Ç–æ–≤ —Å–æ —Å—Ç—Ä–æ–∫–æ–≤—ã–º–∏ –∏–º–µ–Ω–∞–º–∏. –û–Ω–∏ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—Ç ¬´—É—Å—Ç–∞—Ä–µ–≤—à–∏–π¬ª —Å–ø–æ—Å–æ–± —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π –æ–±—ä–µ–∫—Ç–æ–≤.

–ï—Å–ª–∏ –Ω–µ—Ç –º–µ—Ç–æ–¥–∞ `Symbol.toPrimitive`, –¥–≤–∏–∂–æ–∫ JavaScript –ø—ã—Ç–∞–µ—Ç—Å—è –Ω–∞–π—Ç–∏ —ç—Ç–∏ –º–µ—Ç–æ–¥—ã –∏ –≤—ã–∑–≤–∞—Ç—å –∏—Ö —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

-   `toString -> valueOf` –¥–ª—è —Ö–∏–Ω—Ç–∞ —Å–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º ¬´string¬ª.
-   `valueOf -> toString` ‚Äì –≤ –∏–Ω–æ–º —Å–ª—É—á–∞–µ.

–î–ª—è –ø—Ä–∏–º–µ—Ä–∞, –∏—Å–ø–æ–ª—å–∑—É–µ–º –∏—Ö –≤ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –≤—Å—ë —Ç–æ–≥–æ –∂–µ –æ–±—ä–µ–∫—Ç–∞ `user`. –í–æ—Å–ø—Ä–æ–∏–∑–≤–µ–¥—ë–º –µ–≥–æ –ø–æ–≤–µ–¥–µ–Ω–∏–µ –∫–æ–º–±–∏–Ω–∞—Ü–∏–µ–π –º–µ—Ç–æ–¥–æ–≤ `toString` –∏ `valueOf`:
``````

  name: "John",
  money: 1000,

  // –¥–ª—è —Ö–∏–Ω—Ç–∞ —Ä–∞–≤–Ω–æ–≥–æ "string"
  toString() {
    return `{name: "${this.name}"}`;
  },

  // –¥–ª—è —Ö–∏–Ω—Ç–∞ —Ä–∞–≤–Ω–æ–≥–æ "number" –∏–ª–∏ "default"
  valueOf() {
    return this.money;
  }

};

alert(user); // toString -> {name: "John"}
alert(+user); // valueOf -> 1000
