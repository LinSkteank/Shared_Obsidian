–ü–æ—Å–ª–µ–¥—Å—Ç–≤–∏—è —Å–≤–æ–±–æ–¥–Ω–æ–≥–æ `this`

–ï—Å–ª–∏ –≤—ã –¥–æ —ç—Ç–æ–≥–æ –∏–∑—É—á–∞–ª–∏ –¥—Ä—É–≥–∏–µ —è–∑—ã–∫–∏ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è, —Ç–æ–≥–¥–∞ –≤—ã, —Å–∫–æ—Ä–µ–µ –≤—Å–µ–≥–æ, –ø—Ä–∏–≤—ã–∫–ª–∏ –∫ –∏–¥–µ–µ "—Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ `this`" ‚Äì –∫–æ–≥–¥–∞ –º–µ—Ç–æ–¥—ã, –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω—ã–µ –≤–Ω—É—Ç—Ä–∏ –æ–±—ä–µ–∫—Ç–∞, –≤—Å–µ–≥–¥–∞ —Å–æ—Ö—Ä–∞–Ω—è—é—Ç –≤ –∫–∞—á–µ—Å—Ç–≤–µ –∑–Ω–∞—á–µ–Ω–∏—è `this` —Å—Å—ã–ª–∫—É –Ω–∞ —Å–≤–æ–π –æ–±—ä–µ–∫—Ç (–≤ –∫–æ—Ç–æ—Ä–æ–º –±—ã–ª –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –º–µ—Ç–æ–¥).

–í JavaScript `this` —è–≤–ª—è–µ—Ç—Å—è ¬´—Å–≤–æ–±–æ–¥–Ω—ã–º¬ª, –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –≤ –º–æ–º–µ–Ω—Ç –≤—ã–∑–æ–≤–∞ –º–µ—Ç–æ–¥–∞ –∏ –Ω–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —Ç–æ–≥–æ, –≥–¥–µ —ç—Ç–æ—Ç –º–µ—Ç–æ–¥ –±—ã–ª –æ–±—ä—è–≤–ª–µ–Ω, –∞ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —Ç–æ–≥–æ, –∫–∞–∫–æ–π –æ–±—ä–µ–∫—Ç –≤—ã–∑—ã–≤–∞–µ—Ç –º–µ—Ç–æ–¥ (–∫–∞–∫–æ–π –æ–±—ä–µ–∫—Ç —Å—Ç–æ–∏—Ç ¬´–ø–µ—Ä–µ–¥ —Ç–æ—á–∫–æ–π¬ª).

–≠—Ç–∞ –∏–¥–µ—è –≤—ã—á–∏—Å–ª–µ–Ω–∏—è `this` –≤ –º–æ–º–µ–Ω—Ç –∏—Å–ø–æ–ª–Ω–µ–Ω–∏—è –∏–º–µ–µ—Ç –∫–∞–∫ —Å–≤–æ–∏ –ø–ª—é—Å—ã, —Ç–∞–∫ –∏ –º–∏–Ω—É—Å—ã. –° –æ–¥–Ω–æ–π —Å—Ç–æ—Ä–æ–Ω—ã, —Ñ—É–Ω–∫—Ü–∏—è –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–≤—Ç–æ—Ä–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∞ –≤ –∫–∞—á–µ—Å—Ç–≤–µ –º–µ—Ç–æ–¥–∞ —É —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤ (—á—Ç–æ –ø–æ–≤—ã—à–∞–µ—Ç –≥–∏–±–∫–æ—Å—Ç—å). –° –¥—Ä—É–≥–æ–π —Å—Ç–æ—Ä–æ–Ω—ã, –±–æ–ª—å—à–∞—è –≥–∏–±–∫–æ—Å—Ç—å —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –æ—à–∏–±–æ–∫.

–ó–¥–µ—Å—å –º—ã –Ω–µ –±—É–¥–µ–º —Å—É–¥–∏—Ç—å –æ —Ç–æ–º, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ —ç—Ç–æ —Ä–µ—à–µ–Ω–∏–µ –≤ —è–∑—ã–∫–µ —Ö–æ—Ä–æ—à–∏–º –∏–ª–∏ –ø–ª–æ—Ö–∏–º. –ú—ã –¥–æ–ª–∂–Ω—ã –ø–æ–Ω–∏–º–∞—Ç—å, –∫–∞–∫ —Å —ç—Ç–∏–º —Ä–∞–±–æ—Ç–∞—Ç—å, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∞—Ç—å –≤—ã–≥–æ–¥—ã –∏ –∏–∑–±–µ–≥–∞—Ç—å –ø—Ä–æ–±–ª–µ–º.

## [–í–Ω—É—Ç—Ä–µ–Ω–Ω—è—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è: –°—Å—ã–ª–æ—á–Ω—ã–π —Ç–∏–ø](https://learn.javascript.ru/object-methods#vnutrennyaya-realizatsiya-ssylochnyy-tip)

–ü—Ä–æ–¥–≤–∏–Ω—É—Ç–∞—è –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —è–∑—ã–∫–∞

–≠—Ç–æ—Ç —Ä–∞–∑–¥–µ–ª –æ–±—ä—è—Å–Ω—è–µ—Ç —Å–ª–æ–∂–Ω—É—é —Ç–µ–º—É, —á—Ç–æ–±—ã –ª—É—á—à–µ –ø–æ–Ω–∏–º–∞—Ç—å –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∑–∞–ø—É—Ç–∞–Ω–Ω—ã–µ —Å–ª—É—á–∞–∏.

–ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø—Ä–æ–¥–≤–∏–≥–∞—Ç—å—Å—è –±—ã—Å—Ç—Ä–µ–µ, –µ–≥–æ –º–æ–∂–Ω–æ –ø—Ä–æ–ø—É—Å—Ç–∏—Ç—å –∏–ª–∏ –æ—Ç–ª–æ–∂–∏—Ç—å.

–ù–µ–∫–æ—Ç–æ—Ä—ã–µ —Ö–∏—Ç—Ä—ã–µ —Å–ø–æ—Å–æ–±—ã –≤—ã–∑–æ–≤–∞ –º–µ—Ç–æ–¥–∞ –ø—Ä–∏–≤–æ–¥—è—Ç –∫ –ø–æ—Ç–µ—Ä–µ –∑–Ω–∞—á–µ–Ω–∏—è `this`, –Ω–∞–ø—Ä–∏–º–µ—Ä:
``````javascript
let user = {
  name: "–î–∂–æ–Ω",
  hi() { alert(this.name); },
  bye() { alert("–ü–æ–∫–∞"); }
};

user.hi(); // –î–∂–æ–Ω (–ø—Ä–æ—Å—Ç–æ–π –≤—ã–∑–æ–≤ –º–µ—Ç–æ–¥–∞ —Ä–∞–±–æ—Ç–∞–µ—Ç —Ö–æ—Ä–æ—à–æ)

// —Ç–µ–ø–µ—Ä—å –¥–∞–≤–∞–π—Ç–µ –ø–æ–ø—Ä–æ–±—É–µ–º –≤—ã–∑—ã–≤–∞—Ç—å user.hi –∏–ª–∏ user.bye
// –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –∏–º–µ–Ω–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è user.name
(user.name == "–î–∂–æ–Ω" ? user.hi : user.bye)(); // –û—à–∏–±–∫–∞!
``````

–í –ø–æ—Å–ª–µ–¥–Ω–µ–π —Å—Ç—Ä–æ—á–∫–µ –∫–æ–¥–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —É—Å–ª–æ–≤–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä `?`, –∫–æ—Ç–æ—Ä—ã–π –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –∫–∞–∫–æ–π –±—É–¥–µ—Ç –≤—ã–∑–≤–∞–Ω –º–µ—Ç–æ–¥ (`user.hi` –∏–ª–∏ `user.bye`) –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è —É—Å–ª–æ–≤–∏—è. –í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ –±—É–¥–µ—Ç –≤—ã–±—Ä–∞–Ω `user.hi`.

–ó–∞—Ç–µ–º –º–µ—Ç–æ–¥ —Ç—É—Ç –∂–µ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é —Å–∫–æ–±–æ–∫ `()`. –ù–æ –≤—ã–∑–æ–≤ –Ω–µ —Ä–∞–±–æ—Ç–∞–µ—Ç –∫–∞–∫ –ø–æ–ª–æ–∂–µ–Ω–æ!

–í—ã –º–æ–∂–µ—Ç–µ –≤–∏–¥–µ—Ç—å, —á—Ç–æ –ø—Ä–∏ –≤—ã–∑–æ–≤–µ –±—É–¥–µ—Ç –æ—à–∏–±–∫–∞, –ø–æ—Ç–æ–º—É —á—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º `"this"` –≤–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏ —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è `undefined` (–ø–æ–ª–∞–≥–∞–µ–º, —á—Ç–æ —É –Ω–∞—Å —Å—Ç—Ä–æ–≥–∏–π —Ä–µ–∂–∏–º).

–¢–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç (–¥–æ—Å—Ç—É–ø –∫ –º–µ—Ç–æ–¥—É –æ–±—ä–µ–∫—Ç–∞ —á–µ—Ä–µ–∑ —Ç–æ—á–∫—É):

`user.hi();`

–¢–∞–∫ —É–∂–µ –Ω–µ —Ä–∞–±–æ—Ç–∞–µ—Ç (–≤—ã–∑—ã–≤–∞–µ–º—ã–π –º–µ—Ç–æ–¥ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è):

`(user.name == "–î–∂–æ–Ω" ? user.hi : user.bye)(); // –û—à–∏–±–∫–∞!`

–ü–æ—á–µ–º—É? –ï—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –ø–æ–Ω—è—Ç—å, –ø–æ—á–µ–º—É —Ç–∞–∫ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç, –¥–∞–≤–∞–π—Ç–µ —Ä–∞–∑–±–µ—Ä—ë–º—Å—è (–∑–∞–≥–ª—è–Ω–µ–º –ø–æ–¥ –∫–∞–ø–æ—Ç), –∫–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç –≤—ã–∑–æ–≤ –º–µ—Ç–æ–¥–æ–≤ (`obj.method()`).

–ü—Ä–∏—Å–º–æ—Ç—Ä–µ–≤—à–∏—Å—å –ø–æ–±–ª–∏–∂–µ, –≤ –≤—ã—Ä–∞–∂–µ–Ω–∏–∏ `obj.method()` –º–æ–∂–Ω–æ –∑–∞–º–µ—Ç–∏—Ç—å –¥–≤–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:

1.  –°–Ω–∞—á–∞–ª–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä —Ç–æ—á–∫–∞ `'.'` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–≤–æ–π—Å—Ç–≤–æ –æ–±—ä–µ–∫—Ç–∞ ‚Äì –µ–≥–æ –º–µ—Ç–æ–¥ (`obj.method`).
2.  –ó–∞—Ç–µ–º —Å–∫–æ–±–∫–∏ `()` –≤—ã–∑—ã–≤–∞—é—Ç —ç—Ç–æ—Ç –º–µ—Ç–æ–¥ (–∏—Å–ø–æ–ª–Ω—è–µ—Ç—Å—è –∫–æ–¥ –º–µ—Ç–æ–¥–∞).

–ò—Ç–∞–∫, –∫–∞–∫–∏–º –∂–µ –æ–±—Ä–∞–∑–æ–º –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –æ `this` –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –∏–∑ –ø–µ—Ä–≤–æ–π —á–∞—Å—Ç–∏ –≤–æ –≤—Ç–æ—Ä—É—é?

–ï—Å–ª–∏ –º—ã –ø–æ–º–µ—Å—Ç–∏–º —ç—Ç–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤ –æ—Ç–¥–µ–ª—å–Ω—ã–µ —Å—Ç—Ä–æ–∫–∏, —Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ `this`, –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω–æ, –±—É–¥–µ—Ç –ø–æ—Ç–µ—Ä—è–Ω–æ:
``````javascript
let user = {
  name: "–î–∂–æ–Ω",
  hi() { alert(this.name); }
};

// —Ä–∞–∑–¥–µ–ª–∏–º –ø–æ–ª—É—á–µ–Ω–∏–µ –º–µ—Ç–æ–¥–∞ –æ–±—ä–µ–∫—Ç–∞ –∏ –µ–≥–æ –≤—ã–∑–æ–≤ –≤ —Ä–∞–∑–Ω—ã—Ö —Å—Ç—Ä–æ–∫–∞—Ö

hi(); // –û—à–∏–±–∫–∞, –ø–æ—Ç–æ–º—É —á—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º this —è–≤–ª—è–µ—Ç—Å—è undefined
``````
–ó–¥–µ—Å—å `hi = user.hi` —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –∏ –¥–∞–ª–µ–µ –≤ –ø–æ—Å–ª–µ–¥–Ω–µ–π —Å—Ç—Ä–æ–∫–µ –æ–Ω–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è –ø–æ–ª–Ω–æ—Å—Ç—å—é —Å–∞–º–∞ –ø–æ —Å–µ–±–µ, –±–µ–∑ –æ–±—ä–µ–∫—Ç–∞, —Ç–∞–∫ —á—Ç–æ –Ω–µ—Ç `this`.

**–î–ª—è —Ä–∞–±–æ—Ç—ã –≤—ã–∑–æ–≤–æ–≤ —Ç–∏–ø–∞ `user.hi()`, JavaScript –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Ç—Ä—é–∫ ‚Äì —Ç–æ—á–∫–∞ `'.'` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–µ —Å–∞–º—É —Ñ—É–Ω–∫—Ü–∏—é, –∞ —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ ¬´—Å—Å—ã–ª–æ—á–Ω–æ–≥–æ —Ç–∏–ø–∞¬ª, –Ω–∞–∑—ã–≤–∞–µ–º–æ–≥–æ [Reference Type](https://tc39.github.io/ecma262/#sec-reference-specification-type).**

–≠—Ç–æ—Ç —Å—Å—ã–ª–æ—á–Ω—ã–π —Ç–∏–ø (Reference Type) —è–≤–ª—è–µ—Ç—Å—è –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–º —Ç–∏–ø–æ–º. –ú—ã –Ω–µ –º–æ–∂–µ–º —è–≤–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –µ–≥–æ, –Ω–æ –æ–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ —è–∑—ã–∫–∞.

–ó–Ω–∞—á–µ–Ω–∏–µ —Å—Å—ã–ª–æ—á–Ω–æ–≥–æ —Ç–∏–ø–∞ ‚Äì —ç—Ç–æ ¬´—Ç—Ä–∏–ø–ª–µ—Ç¬ª: –∫–æ–º–±–∏–Ω–∞—Ü–∏—è –∏–∑ —Ç—Ä—ë—Ö –∑–Ω–∞—á–µ–Ω–∏–π `(base, name, strict)`, –≥–¥–µ:

-   `base` ‚Äì —ç—Ç–æ –æ–±—ä–µ–∫—Ç.
-   `name` ‚Äì —ç—Ç–æ –∏–º—è —Å–≤–æ–π—Å—Ç–≤–∞ –æ–±—ä–µ–∫—Ç–∞.
-   `strict` ‚Äì —ç—Ç–æ —Ä–µ–∂–∏–º –∏—Å–ø–æ–ª–Ω–µ–Ω–∏—è. –Ø–≤–ª—è–µ—Ç—Å—è true, –µ—Å–ª–∏ –¥–µ–π—Å—Ç–≤—É–µ—Ç —Å—Ç—Ä–æ–≥–∏–π —Ä–µ–∂–∏–º (`use strict`).

–†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –¥–æ—Å—Ç—É–ø–∞ –∫ —Å–≤–æ–π—Å—Ç–≤—É `user.hi` —è–≤–ª—è–µ—Ç—Å—è –Ω–µ —Ñ—É–Ω–∫—Ü–∏—è, –∞ –∑–Ω–∞—á–µ–Ω–∏–µ —Å—Å—ã–ª–æ—á–Ω–æ–≥–æ —Ç–∏–ø–∞. –î–ª—è `user.hi` –≤ —Å—Ç—Ä–æ–≥–æ–º —Ä–µ–∂–∏–º–µ –æ–Ω–æ –±—É–¥–µ—Ç —Ç–∞–∫–∏–º:

`// –∑–Ω–∞—á–µ–Ω–∏–µ —Å—Å—ã–ª–æ—á–Ω–æ–≥–æ —Ç–∏–ø–∞ (Reference Type)
(user, "hi", true)`

–ö–æ–≥–¥–∞ —Å–∫–æ–±–∫–∏ `()` –ø—Ä–∏–º–µ–Ω—è—é—Ç—Å—è –∫ –∑–Ω–∞—á–µ–Ω–∏—é —Å—Å—ã–ª–æ—á–Ω–æ–≥–æ —Ç–∏–ø–∞ (–ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –≤—ã–∑–æ–≤), —Ç–æ –æ–Ω–∏ –ø–æ–ª—É—á–∞—é—Ç –ø–æ–ª–Ω—É—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ–± –æ–±—ä–µ–∫—Ç–µ –∏ –µ–≥–æ –º–µ—Ç–æ–¥–µ, –∏ –º–æ–≥—É—Ç –ø–æ—Å—Ç–∞–≤–∏—Ç—å –ø—Ä–∞–≤–∏–ª—å–Ω—ã–π `this` (`=user` –≤ –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ, –ø–æ `base`).
