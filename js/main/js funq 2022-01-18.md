–ù–µ–±–æ–ª—å—à–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –Ω–µ —Ç–æ–ª—å–∫–æ –æ–±–ª–µ–≥—á–∞—é—Ç —Ç–µ—Å—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ –æ—Ç–ª–∞–¥–∫—É ‚Äì —Å–∞–º–æ —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–Ω–∏–µ —Ç–∞–∫–∏—Ö —Ñ—É–Ω–∫—Ü–∏–π –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Ä–æ–ª—å —Ö–æ—Ä–æ—à–∏—Ö –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–µ–≤!

–ù–∞–ø—Ä–∏–º–µ—Ä, —Å—Ä–∞–≤–Ω–∏–º –Ω–∏–∂–µ –¥–≤–µ —Ñ—É–Ω–∫—Ü–∏–∏ `showPrimes(n)`. –ö–∞–∂–¥–∞—è –∏–∑ –Ω–∏—Ö –≤—ã–≤–æ–¥–∏—Ç [–ø—Ä–æ—Å—Ç–æ–µ —á–∏—Å–ª–æ](https://ru.wikipedia.org/wiki/%D0%9F%D1%80%D0%BE%D1%81%D1%82%D0%BE%D0%B5_%D1%87%D0%B8%D1%81%D0%BB%D0%BE) –¥–æ `n`.

–ü–µ—Ä–≤—ã–π –≤–∞—Ä–∏–∞–Ω—Ç –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –º–µ—Ç–∫—É `nextPrime`:

–í—Ç–æ—Ä–æ–π –≤–∞—Ä–∏–∞–Ω—Ç –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é `isPrime(n)` –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ –Ω–∞ –ø—Ä–æ—Å—Ç–æ–µ:

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –¥–æ–ø—É—Å—Ç–∏–º–æ —Å–æ–∑–¥–∞–≤–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏–∏, –¥–∞–∂–µ –µ—Å–ª–∏ –º—ã –Ω–µ –ø–ª–∞–Ω–∏—Ä—É–µ–º –ø–æ–≤—Ç–æ—Ä–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏—Ö. –¢–∞–∫–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä—É—é—Ç –∫–æ–¥ –∏ –¥–µ–ª–∞—é—Ç –µ–≥–æ –±–æ–ª–µ–µ –ø–æ–Ω—è—Ç–Ω—ã–º.

## Function Expression

–§—É–Ω–∫—Ü–∏—è –≤ JavaScript ‚Äì —ç—Ç–æ –Ω–µ –º–∞–≥–∏—á–µ—Å–∫–∞—è —è–∑—ã–∫–æ–≤–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞, –∞ –æ—Å–æ–±–æ–≥–æ —Ç–∏–ø–∞ –∑–Ω–∞—á–µ–Ω–∏–µ.

–°–∏–Ω—Ç–∞–∫—Å–∏—Å, –∫–æ—Ç–æ—Ä—ã–π –º—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª–∏ –¥–æ —ç—Ç–æ–≥–æ, –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è _Function Declaration_ (–û–±—ä—è–≤–ª–µ–Ω–∏–µ –§—É–Ω–∫—Ü–∏–∏):
```

  alert( "–ü—Ä–∏–≤–µ—Ç" );
}
```

–°—É—â–µ—Å—Ç–≤—É–µ—Ç –µ—â—ë –æ–¥–∏–Ω —Å–∏–Ω—Ç–∞–∫—Å–∏—Å —Å–æ–∑–¥–∞–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–π, –∫–æ—Ç–æ—Ä—ã–π –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è _Function Expression_ (–§—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ–µ –í—ã—Ä–∞–∂–µ–Ω–∏–µ).

–û–Ω–æ –≤—ã–≥–ª—è–¥–∏—Ç –≤–æ—Ç —Ç–∞–∫:

```
  alert( "–ü—Ä–∏–≤–µ—Ç" );
};
```
–í –∫–æ–¥–µ –≤—ã—à–µ —Ñ—É–Ω–∫—Ü–∏—è —Å–æ–∑–¥–∞—ë—Ç—Å—è –∏ —è–≤–Ω–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –∫–∞–∫ –ª—é–±–æ–µ –¥—Ä—É–≥–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ. –ü–æ —Å—É—Ç–∏ –±–µ–∑ —Ä–∞–∑–Ω–∏—Ü—ã, –∫–∞–∫ –º—ã –æ–ø—Ä–µ–¥–µ–ª–∏–ª–∏ —Ñ—É–Ω–∫—Ü–∏—é, —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ, —Ö—Ä–∞–Ω–∏–º–æ–µ –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π `sayHi`.

–°–º—ã—Å–ª –æ–±–æ–∏—Ö –ø—Ä–∏–º–µ—Ä–æ–≤ –∫–æ–¥–∞ –æ–¥–∏–Ω–∞–∫–æ–≤: "—Å–æ–∑–¥–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏—é –∏ –ø–æ–º–µ—Å—Ç–∏—Ç—å –µ—ë –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é `sayHi`".

–ú—ã –º–æ–∂–µ–º –¥–∞–∂–µ –≤—ã–≤–µ—Å—Ç–∏ —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é `alert`:
```

  alert( "–ü—Ä–∏–≤–µ—Ç" );
}
```

## –û—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä —Ä–∞—Å—à–∏—Ä–µ–Ω–∏—è

## [–û—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã (`...`)](https://learn.javascript.ru/rest-parameters-spread-operator#ostatochnye-parametry)

–í—ã–∑—ã–≤–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏—é –º–æ–∂–Ω–æ —Å –ª—é–±—ã–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ–º –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç —Ç–æ–≥–æ, –∫–∞–∫ –æ–Ω–∞ –±—ã–ª–∞ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∞.
``````

``````
–õ–∏—à–Ω–∏–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã –Ω–µ –≤—ã–∑–æ–≤—É—Ç –æ—à–∏–±–∫—É. –ù–æ, –∫–æ–Ω–µ—á–Ω–æ, –ø–æ—Å—á–∏—Ç–∞—é—Ç—Å—è —Ç–æ–ª—å–∫–æ –ø–µ—Ä–≤—ã–µ –¥–≤–∞.

_–û—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã_ –º–æ–≥—É—Ç –±—ã—Ç—å –æ–±–æ–∑–Ω–∞—á–µ–Ω—ã —á–µ—Ä–µ–∑ —Ç—Ä–∏ —Ç–æ—á–∫–∏ `...`. –ë—É–∫–≤–∞–ª—å–Ω–æ —ç—Ç–æ –∑–Ω–∞—á–∏—Ç: ¬´—Å–æ–±–µ—Ä–∏ –æ—Å—Ç–∞–≤—à–∏–µ—Å—è –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –∏ –ø–æ–ª–æ–∂–∏ –∏—Ö –≤ –º–∞—Å—Å–∏–≤¬ª.

–ù–∞–ø—Ä–∏–º–µ—Ä, —Å–æ–±–µ—Ä—ë–º –≤—Å–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã –≤ –º–∞—Å—Å–∏–≤ `args`:

–ú—ã –º–æ–∂–µ–º –ø–æ–ª–æ–∂–∏—Ç—å –ø–µ—Ä–≤—ã–µ –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ, –∞ –æ—Å—Ç–∞–ª—å–Ω—ã–µ ‚Äì —Å–æ–±—Ä–∞—Ç—å –≤ –º–∞—Å—Å–∏–≤.

–í –ø—Ä–∏–º–µ—Ä–µ –Ω–∏–∂–µ –ø–µ—Ä–≤—ã–µ –¥–≤–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–∞ —Ñ—É–Ω–∫—Ü–∏–∏ —Å—Ç–∞–Ω—É—Ç –∏–º–µ–Ω–µ–º –∏ —Ñ–∞–º–∏–ª–∏–µ–π, –∞ —Ç—Ä–µ—Ç–∏–π –∏ –ø–æ—Å–ª–µ–¥—É—é—â–∏–µ –ø—Ä–µ–≤—Ä–∞—Ç—è—Ç—Å—è –≤ –º–∞—Å—Å–∏–≤ `titles`:

–û—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –¥–æ–ª–∂–Ω—ã —Ä–∞—Å–ø–æ–ª–∞–≥–∞—Ç—å—Å—è –≤ –∫–æ–Ω—Ü–µ

–û—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã —Å–æ–±–∏—Ä–∞—é—Ç –≤—Å–µ –æ—Å—Ç–∞–ª—å–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã, –ø–æ—ç—Ç–æ–º—É –±–µ—Å—Å–º—ã—Å–ª–µ–Ω–Ω–æ –ø–∏—Å–∞—Ç—å —á—Ç–æ-–ª–∏–±–æ –ø–æ—Å–ª–µ –Ω–∏—Ö. –≠—Ç–æ –≤—ã–∑–æ–≤–µ—Ç –æ—à–∏–±–∫—É:
``````

  // –û—à–∏–±–∫–∞
``````

–í—Å–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ –ø—Å–µ–≤–¥–æ–º–∞—Å—Å–∏–≤–µ `arguments` –ø–æ–¥ —Å–≤–æ–∏–º–∏ –ø–æ—Ä—è–¥–∫–æ–≤—ã–º–∏ –Ω–æ–º–µ—Ä–∞–º–∏.

–†–∞–Ω—å—à–µ –≤ —è–∑—ã–∫–µ –Ω–µ –±—ã–ª–æ –æ—Å—Ç–∞—Ç–æ—á–Ω—ã—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤, –∏ –ø–æ–ª—É—á–∏—Ç—å –≤—Å–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ –º–æ–∂–Ω–æ –±—ã–ª–æ —Ç–æ–ª—å–∫–æ —Å –ø–æ–º–æ—â—å—é `arguments`. –≠—Ç–æ—Ç —Å–ø–æ—Å–æ–± –≤—Å—ë –µ—â—ë —Ä–∞–±–æ—Ç–∞–µ—Ç, –º—ã –º–æ–∂–µ–º –Ω–∞–π—Ç–∏ –µ–≥–æ –≤ —Å—Ç–∞—Ä–æ–º –∫–æ–¥–µ.

–ù–æ —É –Ω–µ–≥–æ –µ—Å—Ç—å –æ–¥–∏–Ω –Ω–µ–¥–æ—Å—Ç–∞—Ç–æ–∫. –•–æ—Ç—è `arguments` –ø–æ—Ö–æ–∂ –Ω–∞ –º–∞—Å—Å–∏–≤, –∏ –µ–≥–æ —Ç–æ–∂–µ –º–æ–∂–Ω–æ –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å, —ç—Ç–æ –≤—Å—ë –∂–µ –Ω–µ –º–∞—Å—Å–∏–≤. –û–Ω –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –º–µ—Ç–æ–¥—ã –º–∞—Å—Å–∏–≤–æ–≤, –ø–æ—ç—Ç–æ–º—É –º—ã –Ω–µ –º–æ–∂–µ–º, –Ω–∞–ø—Ä–∏–º–µ—Ä, –≤—ã–∑–≤–∞—Ç—å `arguments.map(...)`.

–ö —Ç–æ–º—É –∂–µ, `arguments` –≤—Å–µ–≥–¥–∞ —Å–æ–¥–µ—Ä–∂–∏—Ç –≤—Å–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ ‚Äî –º—ã –Ω–µ –º–æ–∂–µ–º –ø–æ–ª—É—á–∏—Ç—å –∏—Ö —á–∞—Å—Ç—å. –ê –æ—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –ø–æ–∑–≤–æ–ª—è—é—Ç —ç—Ç–æ —Å–¥–µ–ª–∞—Ç—å.

–°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –¥–ª—è –±–æ–ª–µ–µ —É–¥–æ–±–Ω–æ–π —Ä–∞–±–æ—Ç—ã —Å –∞—Ä–≥—É–º–µ–Ω—Ç–∞–º–∏ –ª—É—á—à–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã.

–°—Ç—Ä–µ–ª–æ—á–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –Ω–µ –∏–º–µ—é—Ç `"arguments"`

–ö–∞–∫ –º—ã –ø–æ–º–Ω–∏–º, —É —Å—Ç—Ä–µ–ª–æ—á–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π –Ω–µ—Ç —Å–æ–±—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ `this`. –¢–µ–ø–µ—Ä—å –º—ã –∑–Ω–∞–µ–º, —á—Ç–æ –Ω–µ—Ç –∏ —Å–≤–æ–µ–≥–æ –æ–±—ä–µ–∫—Ç–∞ `arguments`.

## [–û–ø–µ—Ä–∞—Ç–æ—Ä —Ä–∞—Å—à–∏—Ä–µ–Ω–∏—è](https://learn.javascript.ru/rest-parameters-spread-operator#spread-operator)

–ò —Ç—É—Ç –Ω–∞–º –ø–æ–º–æ–∂–µ—Ç _–æ–ø–µ—Ä–∞—Ç–æ—Ä —Ä–∞—Å—à–∏—Ä–µ–Ω–∏—è_. –û–Ω –ø–æ—Ö–æ–∂ –Ω–∞ –æ—Å—Ç–∞—Ç–æ—á–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã ‚Äì —Ç–æ–∂–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç `...`, –Ω–æ –¥–µ–ª–∞–µ—Ç —Å–æ–≤–µ—Ä—à–µ–Ω–Ω–æ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ–µ.

–ö–æ–≥–¥–∞ `...arr` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –ø—Ä–∏ –≤—ã–∑–æ–≤–µ —Ñ—É–Ω–∫—Ü–∏–∏, –æ–Ω ¬´—Ä–∞—Å—à–∏—Ä—è–µ—Ç¬ª –ø–µ—Ä–µ–±–∏—Ä–∞–µ–º—ã–π –æ–±—ä–µ–∫—Ç `arr` –≤ —Å–ø–∏—Å–æ–∫ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤.

–≠—Ç–∏–º –∂–µ —Å–ø–æ—Å–æ–±–æ–º –º—ã –º–æ–∂–µ–º –ø–µ—Ä–µ–¥–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤:

## [–ò—Ç–æ–≥–æ](https://learn.javascript.ru/rest-parameters-spread-operator#itogo)

# –§—É–Ω–∫—Ü–∏–∏-—Å—Ç—Ä–µ–ª–∫–∏

‚Ä¶–¢–∞–∫–æ–π –∫–æ–¥ —Å–æ–∑–¥–∞—ë—Ç —Ñ—É–Ω–∫—Ü–∏—é `func` —Å –∞—Ä–≥—É–º–µ–Ω—Ç–∞–º–∏ `arg1..argN` –∏ –≤—ã—á–∏—Å–ª—è–µ—Ç `expression` —Å –ø—Ä–∞–≤–æ–π —Å—Ç–æ—Ä–æ–Ω—ã —Å –∏—Ö –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º, –≤–æ–∑–≤—Ä–∞—â–∞—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

–î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, —ç—Ç–æ –±–æ–ª–µ–µ –∫–æ—Ä–æ—Ç–∫–∏–π –≤–∞—Ä–∏–∞–Ω—Ç —Ç–∞–∫–æ–π –∑–∞–ø–∏—Å–∏:

–¢–æ –µ—Å—Ç—å, `(a, b) => a + b` –∑–∞–¥–∞—ë—Ç —Ñ—É–Ω–∫—Ü–∏—é —Å –¥–≤—É–º—è –∞—Ä–≥—É–º–µ–Ω—Ç–∞–º–∏ `a` –∏ `b`, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–∏ –∑–∞–ø—É—Å–∫–µ –≤—ã—á–∏—Å–ª—è–µ—Ç –≤—ã—Ä–∞–∂–µ–Ω–∏–µ —Å–ø—Ä–∞–≤–∞ `a + b` –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –µ–≥–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

-   –ï—Å–ª–∏ —É –Ω–∞—Å —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω –∞—Ä–≥—É–º–µ–Ω—Ç, —Ç–æ –∫—Ä—É–≥–ª—ã–µ —Å–∫–æ–±–∫–∏ –≤–æ–∫—Ä—É–≥ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –º–æ–∂–Ω–æ –æ–ø—É—Å—Ç–∏—Ç—å, —Å–¥–µ–ª–∞–≤ –∑–∞–ø–∏—Å—å –µ—â—ë –∫–æ—Ä–æ—á–µ:

–§—É–Ω–∫—Ü–∏–∏-—Å—Ç—Ä–µ–ª–∫–∏ –º–æ–≥—É—Ç –±—ã—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω—ã —Ç–∞–∫ –∂–µ, –∫–∞–∫ –∏ Function Expression.

–ù–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–æ–≥–æ —Å–æ–∑–¥–∞–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–∏:

–ü–æ–Ω–∞—á–∞–ª—É —Ñ—É–Ω–∫—Ü–∏–∏-—Å—Ç—Ä–µ–ª–∫–∏ –º–æ–≥—É—Ç –ø–æ–∫–∞–∑–∞—Ç—å—Å—è –Ω–µ–æ–±—ã—á–Ω—ã–º–∏ –∏ —Ç—Ä—É–¥–Ω–æ—á–∏—Ç–∞–µ–º—ã–º–∏, –Ω–æ —ç—Ç–æ –±—ã—Å—Ç—Ä–æ –ø—Ä–æ–π–¥—ë—Ç, –∫–∞–∫ —Ç–æ–ª—å–∫–æ –≥–ª–∞–∑–∞ –ø—Ä–∏–≤—ã–∫–Ω—É—Ç –∫ —ç—Ç–∏–º –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è–º.

## [–ú–Ω–æ–≥–æ—Å—Ç—Ä–æ—á–Ω—ã–µ —Å—Ç—Ä–µ–ª–æ—á–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏](https://learn.javascript.ru/arrow-functions-basics#mnogostrochnye-strelochnye-funktsii)
